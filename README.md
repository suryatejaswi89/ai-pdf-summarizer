# AI PDF Summarizer

This project extracts text from PDF files and generates summaries using AI.

## Features
- Extract text from PDFs
- Generate summaries using LLMs
- Simple and fast

## Tech Stack
- Python
- OpenAI API
- PyPDF

## How to Run
1. Install dependencies:
   pip install openai pypdf

2. Add your API key

3. Run:
   python pdf_summarizer.py

## Use Cases
- Business reports
- Research papers
- Resume summarization
